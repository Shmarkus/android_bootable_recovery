Welcome to CWM Rockchip port

This project is modification of stock ClockWorkMod for Rockchip devices. Since RK devices can not use IOCTL MEMERASE properly for some reason a workaround is needed. I merged this workaround from stock recovery found on radxa github page to recovery used in CyanogenMod.

Right now only cm10.1 and jellybean-stable branches are updated, although this update is easy to apply on other branches when necessary!

Usage

  • Fetch CM from Github (plenty of examples in the internet).
  • edit manifest file so that this recovery is used instead of CyagenMods default recovery
  • Build and have fun doing backups and flashing new ROMs
